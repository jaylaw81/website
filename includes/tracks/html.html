HTML

It starts with clean markup

HTML is a coding language that's used to create web pages that can be posted on the Internet. HTML stands for Hyper Text Markup Language. You use HTML to tell a website how to present text, images and videos. HTML "tags" all of these design elements with codes that instruct the web browser how to display the elements. The tags also provide layout and formatting information so that the web page will look the way its designer intended.
