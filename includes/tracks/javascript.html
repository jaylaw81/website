JavaScript

Interactivity can be awesome

JavaScript is a programming language designed to add interactivity to web pages and create web applications. JavaScript can interact with HTML source code, enabling web developers to spice up their sites with dynamic content. Depending on the developer's intent, JavaScript code may run when the user opens the web page, clicks or drags some page element with the mouse, types something on the keyboard, submits a form, or leaves the page.
