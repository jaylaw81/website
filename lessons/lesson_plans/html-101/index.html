Intro into HTML

HTML
HyperText Markup Language is a markup language that web browsers use to interpret and compose text, images and other material into visual or audible web pages.

History of HTML

Why we use HTML

HTML is the skeleton of a webpage. It is the basis in which everything else you see on a web page exists.

If we were to remove the skeleton, the web page would fall apart.

Web browsers

Web browsers interpret and display the HTML tags on a web page and display it back to the user.

Types of Browsers

There are alot of browsers available, however in terms of web design and development we only concern ourselves with a few.

HTML Attributes

  • HTML elements can have attributes.
  • Attributes provide additional information about an element.
  • Attributes are always specified in the start tag.
  • Attributes come in name/value pairs like: name="value"
