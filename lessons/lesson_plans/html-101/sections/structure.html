Now that we've built our first page lets talk about all of the elements or tags that make up the structure of our page.

  • The first tag is the DOCTYPE declaration. This communicates to the browser what version of html the page is written in. Please note that this tag does not have a closing tag.
  • Next is our html tag which tells the browser that this is an html document. The html tag contains all of the other html elements aside from the DOCTYPE declaration. You'll notice that at the very end of the code is the html closing tag.
  • Then we have the head tag. As you can see inside the head element we have the title element which contains the required page title. The head element can also contain meta data which is information about our page, links to css style sheets which allow us to add styling and color to the page, also we can link scripts to our page to add interactivity, animation, error checking and many other features.
  • After that we have the body tag. The body tag contains all of the contents of the html document, text, images , videos, lists, etc.
  • finally, you'll see the h1 tag which is a heading tag, inside this tag we've placed our Hello world text.

HTML Attributes
  • HTML elements can have attributes.
  • Attributes provide additional information about an element.
  • Attributes are always specified in the start tag.
  • Attributes come in name/value pairs like: name="value"
