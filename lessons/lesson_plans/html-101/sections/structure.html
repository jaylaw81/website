Now that we've built our first page lets talk about all of the elements or tags that make up the structure of our page.

<0l>
  • The <!DOCTYPE html> declaration tag. This communicates to the browser what version of html the page is written in. Please note that this tag does not have a closing tag.
  • The <html> tag which tells the browser that this is an html document. The html tag contains all of the other html elements aside from the DOCTYPE declaration. You'll notice that at the very end of the code is the html closing tag.
  • The <head> tag. The head element contains the page <title>, <meta> data (which is information about our page), links to css style sheets, and scripts.
  • The <body> tag contains all of the contents of the html document, text, images , videos, lists, etc.
  • The <h1> tag is a heading tag which contains our Hello world text.

    • HTML Attributes
    • HTML elements can have attributes.
    • Attributes provide additional information about an element.
    • Attributes are always specified in the start tag.
    • Attributes come in name/value pairs like: name="value"
