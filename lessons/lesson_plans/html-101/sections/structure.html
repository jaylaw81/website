Basic HTML Structure

HTML is the structure of a webpage. Lets learn the essential elements and basic structure of a web page by creating a Hello World page.
This is a common first exercise when learning any programming language. Below you'll notice the code sample of our page and the tags that make up this very basic page.

Before we buil our first page lets talk about all of the elements that make up the structure of our page.

  1. The <!DOCTYPE html> declaration tag. This communicates to the browser what version of html the page is written in. Please note that this tag does not have a closing tag.
  2. The <html> tag which tells the browser that this is an html document. The html tag contains all of the other html elements aside from the DOCTYPE declaration. You'll notice that at the very end of the code is the html closing tag.
  3. The <head> tag. The head element contains the page <title>, <meta> data (which is information about our page), links to css style sheets, and scripts.
  4. The <body> tag contains all of the contents of the html document, text, images , videos, lists, etc.
  5. The <h1> tag is a heading tag which contains our Hello world text.

What is syntax?

Syntax
is the set of rules that a programming language needs to use in order for the computer to interpret them properly. Machines are not flexible and can only interpret the code correctly when it's in the expected form or syntax.

HTML Element Syntax

The elements of an HTML document need to be structured with the proper syntax in order for the page to be viewed correctly.

  • An HTML element starts with a start tag / opening tag : <p>
  • An HTML element ends with an end tag / closing tag : </p>
  • The element content is everything between the start and the end tag
  • Some HTML elements have empty content
    • Empty elements are closed in the start tag : <br />
  • Most HTML elements can have attributes which we'll learn about later

Exploring Element Attributes

HTML elements can have attributes, which provide additional information about the element. Atrributes are always contained within the starting tag of an element. Let's look at a couple of elements that contain attributes.


Our first example is the image tag, which contains a number of attribute tags that tell the browser different thinga about the image element.

    Image Attributes
  • Width and Height - tells the browser the height and width of the image.
  • Title - acts as a caption for the image and is displayed on hover.
  • Alt - provides alternate text when the image element cannot be displayed. This element is also used by screen readers to read content to users that are visiually impaired.
  • Anchor tag Attributes
  • href Provides the destination URL or location for the anchor tag
  • target Provides the target for the link will it open in the same tab or window using target="_self"? Or will it open in a new tab or window, target="_blank"

What's a Meta tag and why are they important?

Metadata
Metadata is "data about data". So that means we have descriptive information about our data object. For instance, contained within a .jpg or image file is more information about the picture like the date, time, and location where the image was taken. In the case of a website we use metadata or<meta> tags to describe our website and provide information about it. This helps search engines and users find our site. Some common meta data attributes title, description, keywords and author. Let's take the time to fill out this information in your html template.
