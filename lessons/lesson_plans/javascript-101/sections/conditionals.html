Conditional - If/Else Statements

If raining take an umbrella, Else put on sunscreen.

The if statement is used to specify a condition and then run a block of code. This can be used by itself or with an else block that stipulates a code block to execute in case the first condition is not met.

Do it yourself!

  1. Open your javascript file.
  2. Place the if/else example into your file.
  3. Change the name variable and age to see the different results in your page.
