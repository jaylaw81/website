CSS Structure

CSS Back Story

CSS (Cascading Style Sheet) is code used to control the visible and invisible properties of the elements and content in your page. It is commonly used to control the page's color scheme, the structural layout of the page and shape of elements. It can also be used to control properties of fonts (uniquely created alphabet characters) and the different states of an anchor. There are many advanced ways to use CSS that you may learn in your journey of learing web design and development.

A part of understanding the structure is also understanding the page. The same way you would read an email, the browser also reads the code in a page - From top to bottom. Generally a web page will have three parts to it anatomy - Header area, Body area and Footer area. This is the same way the broswer reads a CSS file.

In many of our discussions about code we have mentioned the parent and child relationships between elements. With this thought you can understand that from parent to child there can also be inheritance - Like eye color. If we set the "color" property of blue "#0000ff" to the <body> element, then all the text in the page would inherit the color and turn blue.

How CSS selectors work

A CSS rule set consists of a selector and a declaration block:

  • The selector points to the HTML element you want to style.
  • The declaration block contains one or more declarations separated by semicolons.
  • Each declaration includes a property name and a value, separated by a colon.

Units of Measure - Commonly Used in Web Pages

When controlling the shape and size of elements on the page there are Unites of Measure available for you to use. Below are the three most commonly used units.

  • % | Percentage - Usually used to control the percentage of an element's width or height of it's parent width or height.
  • px | pixels (a dot on the computer screen) - Screen resolution is generally 32 pixels per inch.
  • em | 1em is equal to the current font size. 2em means 2 times the size of the current font. E.g., if an element is displayed with a font of 12 pt, then '2em' is 24 pt. The 'em' is a very useful unit in CSS, since it can adapt automatically to the font that the reader uses
