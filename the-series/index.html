The Series

Having fun is priority numero uno.

Held the first 3 weeks of every month, "The Basics" Series is broken down into smaller components called Tracks: HTML, CSS and JavaScript. In these tracks, will be classes/sessions where students will be taught the fundamentals and best practices in each Track. We'll start with the basics in laying out HTML, then learn how to style the HTML using Cascading Style Sheets (CSS) and interact with the HTML using JavaScript.

Classes are held for 3 weeks, on Saturdays, from 9:00am to 2:00pm

Week 1: HTML

Week 2: CSS

Week 3: JavaScript

Students will be provided computers. Lunch will not be provided, so students will need to bring a lunch with them
